Components

For certain functions in EPiServer you need to install components on the computer on which you are working. Install these by downloading them from the Internet. The functions that require client programs are:

  • Spell-checking, Microsoft Office
    A component that uses spell-checking in Microsoft Word from the editor
  • Export to Excel, Microsoft Office
    A component that can export formula data to Microsoft Excel
  • Advanced file uploading
    A component that supports advanced file uploading by dragging and dropping, for example
    Select the Components tab on the first page in Editor mode. You can find it by clicking on the Root page in the structure.

The list shows the installed components. Those ticked green are installed on your machine. Those crossed red are not installed. Select Install if you want to install the client software, and follow the instructions in the dialog boxes.

You will need to install some ActiveX components on your computer. If you have Windows XP with Service Pack 2 you must approve this installation separately.
