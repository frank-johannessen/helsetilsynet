Edit User

All persons who in any way are, or have been, logged onto the website are stored in EPiServer. Some of them are created in EPiServer, others in Windows. When you create a user, or the users create themselves through self-registration, their details are entered. To edit these details later, search for the user under Search user/group and then edit the user. All persons with access to Edit mode can modify some of their details under My Settings. You can modify more information from Admin mode, such as username and password, when the account should be active, the group in which the user is a member etc.

These properties can only be modified for the users that have been created through self-registration or via Create user. These users are marked with a globe symbol in the search result. Modifying this type of information for a Windows user has to be done in Windows. From Admin mode you can modify the same details for a Windows user as the user can themselves via Edit mode.

Create User

Using the option Create user you can set up users in EPiServer, who do not exist in Windows. This option is used in most cases when setting up an extranet and you want to grant permissions to people who are not usually accessible in the Windows system. Enter the information about the user and select which group or groups he/she is going to belong to and click on Add user.

Display options

The Display option is a new function for EPiServer 4.50. It allows the editor to select whether edit mode should be extended or not. In version 4.50, there are several extra buttons in the toolbar in edit mode. Displaying the extra buttons is a personal choice. Which symbols are displayed in the tree structure in edit mode is also a personal choice. Editors can change the settings themselves under My settings. These settings can also be made when the account is created.

All editors are given the extended toolbar and all symbols in the tree structure as default setting. A default setting is a setting that can be reset in web.config.
