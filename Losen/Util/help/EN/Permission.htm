Permissions for Functions

EPiServer’s Edit mode has several functions that allow you to limit the access for certain people or groups. Most functions are connected to the Link tool where the editors can upload images and documents. The following functions can be access protected.

All functions available in the Editor gives one or more groups or persons access to all buttons and functions in Edit mode. You might want to hide certain buttons in certain templates. This command allows you to allocate all the buttons to certain persons or groups. They then get access to all the buttons while others only get the buttons they are allocated for each template.

Detailed error message for troubleshooting  provides one or more groups/individuals with access to detailed error messages. In system settings, it is possible to activate a function that provides visitors with a form to fill in whenever there is a technical error on the website. By changing the access rights here, you can specify who should receive these forms.

Show and publish pages from Microsoft Office Add-in makes it possible for one or more users/groups to publish pages from Microsoft Office. This function requires that an add-in program is installed on the computer being used. Read more about this in the Editor's Manual.

Allow advanced upload of files using for example drag-and-drop makes it possible to allow a group or individual to use advanced uploading. This means, for example, that an editor can upload several files at a time by dragging and dropping files from Window’s file manager to EPiServer’s file manager

Adding/modifying permissions to a function for a group or user

  1. Select Modify for the function you want to modify.
  2. Existing groups or users with access appear in a list
  3. Select Add group if you want to add a whole group with access to this function. Select Add userif you want to add a person with access to this function.
  4. The groups and persons in the system appear in the window that opens. Double-click on the name to add the group or user.
  5. Select OK.
  6. The group or user appears in the list with its check box selected. This means that the group or user has access to this function.
  7. Select Save.

Delete permissions to a function for a group or user

  1. Select Modify for the function you want to modify.
  2. Existing groups or users with access appear in a list
  3. Tick the check box after the group or user for which you want to remove access. The check box is then deselected and the tick in the box disappears.
  4. Select Save.

Accessing functions in the link tool

A new link is tool included in version 4.50 of EPiServer. The link tool is the tool that opens when an editor is to upload an image or a document. What the editor can do and not do in the tool is set in the file manager via access rights. Which link tool to use is a configuration question. Read more in web.config Settings. This means that you can upgrade to 4.50 without having to use the new tool. You were are also able to work with directory access rights in previous versions of EPiServer. Many versions have given the editor access to different tools in the link tool. In the older tool there are buttons used to create directories, change names and remove directories and files. It is possible to remove these functions for groups or individuals if required. It is possible for the following functions to be access protected.

Create folder in link tool Gives one or more groups or persons the right to create directories in the link tool

Delete file in link tool Gives one or more groups or persons the right to remove files from the linktool

Delete folder in link tool Gives one or more groups or persons the right to delete directories from the linktool

Rename file/folder in link tool Gives one or more groups or persons the right to change the name of files and directories in the link tool

Upload file to page specific folder Gives one or more groups or persons the right to upload files to the page directories in the link tool. A directory is then linked to the page where all the files are stored. The directory name is made up of digits.

Upload file to common folder Gives one or more groups or persons the right to upload files to the directory or shared directories in the link tool
