Microsoft Office Add-In

If you want to publish information from Microsoft Office directly to EPiServer you need to install a client software on your computer. The installation is carried out by downloading the software from the internet. You also need to activate the website that you want to publish to. The client software gives you the choice Publish to EPiServer in MS Word, Excel and PowerPoint.

Select the Microsoft Office Addin tab on the first page of the editor mode. This can be found by clicking the page Root in the structure. You can se which version of the client tool you have installed. If you don't have the latest version the button Install will be activated. Choose to install the client software and follow the instructions in the dialogue boxes displayed.

On the tab Activate you can choose to activate the web site as target for your Office publishing. It is possible to publish from Office to many different web sites. Choose the button Activate on each web site that is a target web site.
