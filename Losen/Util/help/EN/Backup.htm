Backup

With the Backup command, you can make a backup copy of your database and view a history of the copies that have been made previously. If you are using SQL Server as your database, there are better backup tools there. 

Remember that this backup only creates a copy of the database and stores it on the same hard drive as the original database. In order to restore the copied database in the event of a disk crash, you should always copy the copy to a 'safe' place.
