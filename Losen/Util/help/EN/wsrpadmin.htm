WSRP Sources

EPiServer Portal Framework is required to consume portlets in EPiServer. The standard EPiServer installation contains a demo example. Contact your supplier if you wish to use EPiServer Portal Framework.

To consume portlets in EPiServer, you need to set up the different sources that will be available in Admin mode.

How to set up a WSRP source:

  1. Select WSRP Sources from the Config tab.
  2. Click New. If you want to change an existing source, click the source in question.
  3. Enter a name for the source in the Name field. This name will be displayed for the portal users.
  4. State the addresses to the source in the remaining four fields. This information is provided by the WSRP producer.
  5. Click Save.
