Edit Categories

Categories in an EPiServer Web site can be used in various ways. It is common to categorize all information in order to achieve an advanced search or to achieve a personalized list of the most interesting pages. Categories are used away from all Web sites, but those categories that the editor may use are defined using the Edit categories function. It is possible to specify categories that are to be visible to and selected by the editor, here. If the category cannot be selected, it will only appear as a heading.

Add a new category as follows:

  1. Select Edit categories on the Admin tab.
  2. Select Add to add new main categories.
  3. Specify a name in the Name field. This name is used when programming.
  4. Enter a description in the Description field. This name is seen by the editors when selecting category. These fields can also be language encoded.
  5. Select Save.

Add a category below the selected category. Note that no changes will be made until you press Save after the category.
