File version management

In the file manager you can track different versions of the file using the version manager. You can see the history of the file, when it was created, by whom and any comments. As editor you can check documents in and out that you are working on so that nobody can modify the file at the same time as you. Access version management through the file manager menu that opens when you right-click a file.
If you cannot see the functions, this may be because version management is set to be active only in certain places in the folder structure. Check this with your system administrator.

Checking files in and out

When you check out a file it is locked so that nobody can modify it at the same time as you. However, despite it being locked, other editors or visitors to your Web site can see the content.
When you have made your changes and want to make the file accessible to others, check it in in the file manager. You can also add a comment in which you briefly describe the changes that you have made. Comments make it easier for other editors to understand the content of the latest version and the changes that have been made.

Integrated with Microsoft Office

Version management is associated with the Office programs, Word, Excel and PowerPoint, and allows you to open files in these programs directly from the file manager. When you have made your changes you can check in the files and make comments on your changes directly from the Office programs.

Working with version management through Office programs

  1. Right-click the file you want to edit.
  2. Select Check out. Your user name is displayed in the Checked out by column indicating that the file is locked by you.
  3. Right-click on the file again. Select Edit in Office. If you have selected a Word file to edit, Microsoft Word opens automatically.
  4. Make your changes in the document. Select Check in. A dialog box is displayed, in which you can enter a comment about your changes.
  5. Enter your comment and select OK. A new version has been created and is available for editors and visitors to the Web site.

Undoing Check out

If you do not make any changes to the document that you have checked out, restore it by right-clicking on the file you checked out and select Undo Check out.

Working with version management on other files

You can also carry out version management on files that cannot be edited with Office programs.
  1. Open the file by double-clicking it. The relevant program is opened and you can make changes to the file. Save the file locally on your computer by selecting Save as...
  2. Right-click the selected file and select Create new version.
  3. Select a new version by selecting Browse. Select the file you have edited and saved on your computer.
  4. Select Open.
  5. Enter your comment and select OK. A new version of the file has been created and is available for editors and visitors to the Web site.

Version history

Information about all updated versions of a checked in file are available in version history. You can see who has made the changes, when they were made and also see a comment about the changes. Right-click the file you want to see the history of. Select Version history.

Downloading an earlier version

All versions of the file are stored. This means that you can download and work on earlier versions.
  1. Right-click the file you want to see an earlier version of. Select Version history.
  2. Right-click the version you want to look at. Select Download. The file is opened in the relevant program.

Republishing an earlier version

As all earlier versions of the file are stored you can restore them, republishing earlier versions.
  1. Right-click the file you want to see an earlier version of. Select Version history.
  2. Right-click the version you want to republish. Select Restore version. The restored version is available for editors and visitors to the Web site.

Deleting versions

  1. Right-click the file you want to see an earlier version of. Select Version history.
  2. Right-click the version you want to delete. Select Delete, and the version is immediately deleted.
  3. You can only delete earlier versions, unpublished versions.
