Remote websites

The remote website function is to enable the creation of load balanced websites and the transfer of data from one EPiServer website to another. Load balanced websites are common in large organisations, where there is a considerable need for accessibility and numerous simultaneous visitors. To be able to take information from one website to another, is a function that is often requested by customers with several websites on EPiServer on several different machines, such as a public website and an intranet, etc. Editors need only create information in one place, but it is displayed in all.
As the administrator in the standard version, you can set this up yourself without needing to develop and adapt your website. More information on this can be found at www.episerver.com.
