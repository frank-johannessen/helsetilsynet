Licence information

The licence information for your EPiServer website is stored in an encrypted file on the web server. This information can tell you to whom the licence belongs, what type of licence it is and if there are any relevant restrictions. If, for any reason, you need to change your licence, if, for example, you change the ip-address or type of licence, etc., you can upload and install the new license file from the Install a new licence folder.
