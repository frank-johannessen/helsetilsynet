Conflict Editor

If two people edit a page simultaneously, the system will warn the users of a conflict in connection with the publishing of the page. This will occur if a person has edited a page at the same time as someone else. The first person to publish the page will not receive a warning. However, the second person will receive a warning when he or she publishes the page. A window appears and the editor can select which of the two versions should be published.
