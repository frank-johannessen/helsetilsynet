Dynamic Properties on Page

Using the Dynamic properties tool, it is possible to customise functions in the website so that they apply to a particular page and its sub-pages. For example, under a branch in the tree, you can determine from where the menus are to be seen. By creating new dynamic properties, you can tailor the structure and functionality of either the entire or parts of the website.

The different dynamic properties that are present vary considerably depending on which website you have built up and which functions you want. For example, you can set which standard search words are to apply for a page. This is used to make it easier for search engines to find the page and index it in the most appropriate way. These search words may need to be different depending on which page they relate to. The page about our products should perhaps have different search words from the page containing contact details. By entering a few search words and choosing Save, you can save the particular search words for the page in question. If you click in the check box beside the field, the search words you have specified are also saved for all the sub-pages below the page you have selected.

To find out which values apply to a particular page, you can point to the various symbols by the input box under dynamic properties and find out about the value by means of help texts.

* The value in this field is defined on the page you have selected and is displayed in the field.

? The value in this field has no value.

^ The value in this field is defined on a page higher in the structure and this page has inherited it. The value is displayed in a help box by pointing to the symbol.

If you choose to save a new value for a dynamic property, the value is saved for the page you selected before you chose Dynamic properties. If you also want the value to apply to all sub-pages of the selected page and override all earlier properties, select the check box to the left of the input field.
