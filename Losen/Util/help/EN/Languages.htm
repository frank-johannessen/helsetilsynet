Edit Languages

EPiServer is capable of managing all types of language. There are eight languages as standard, Swedish, Norwegian, Danish, Finnish, German, French, Spanish and English. If your organization requires further languages, it is possible to add these. By adding new languages under Edit language, you can add extra support for language encoding of an individual page. If a page has been language encoded, all standard texts on the page are in the selected language, provided the language has been translated. If you also require local settings such as time format, currency etc. to be displayed in the applicable language, it must be listed under Edit language.
