Search User/Groups

Using the Search user/group command, you can search for the users and groups in EPiServer and change the settings for each user or group. Enter a name or choose a type and press Search. If you want to list all groups and users, leave the name field empty and the type All and choose Search.

The search results are listed, and you can then change the information about the group/user by clicking on the group or user.
