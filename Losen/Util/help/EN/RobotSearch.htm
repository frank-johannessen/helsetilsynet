Robot Search

Most Internet search engines search Web sites for information and then store the information in their databases. This is known as indexing Web sites. Most search engines cannot index Web sites when the address includes a question mark (?). By default, all EPiServer page addresses contain a question mark and a unique number. You must adjust the page addresses to allow search engines to index the Web site automatically.

If you want Internet search engines to be able to index your Web site, mark the option Use links which allow search engines to index the Web site, and then select OK.
If you now look at the address for a specific page, EPiServer has changed it from www.foretaget.se/templates/page.aspx?id=1234 to www.foretaget.se/templates/page____1234.aspx. This shows that the page can be indexed.
