My settings

The first time you go into edit mode, you should begin by entering your personal settings. This is done on the My settings tab. You can find it by clicking on the Root page in the structure, or on My settings in the toolbar.

User information

This is primarily a matter of identifying yourself in the system. By filling in your personal details such as e-mail address etc. on the User information tab, the system can communicate with you.

Display options

Customize the toolbar on the Display options tab.
If you select Use default settings the toolbar will display all available tools. To limit your toolbar check the Use default settings checkbox and then select Limit toolbar in the menu.
You can also select the symbols visible in the tree structure by checking or unchecking the Symbols in the tree structure checkboxes.
