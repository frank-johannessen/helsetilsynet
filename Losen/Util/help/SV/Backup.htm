Backup

Med alternativet Backup kan du göra en säkerhetskopia av din databas samt se en historik över de kopior som gjorts tidigare. Använder du SQL Server som databas finns bättre verktyg för backuper där.

Tänk på att denna backup bara skapar en kopia av databasen och lagrar den på samma hårddisk som originaldatabasen. För att kunna återföra den kopierade databasen vid eventuell diskkrasch bör du därför alltid kopiera kopian till en "säker" plats.
