Robotsökning

De flesta sökomotorer på internet söker av webbplatser efter information och lagrar informationen i sina databaser. Man brukar kalla det att indexera webbplatser. De flesta sökmotorer klarar inte av att indexera webbplatsen där adressen innehåller frågetecken (?). Som standard innehåller alla adresser till sidor i EPiServer ett frågetecken och likamed ett unikt nummer. För att sökmotorer skall kunna indexera webbplatsen med automatik är det därför viktigt att vi ställer om adresserna så sökmotorerna kan indexera sidan.

Om du vill att sökmotorer på internet skall kunna indexera er webbplats markera alternativet Använd länkar som möjliggör för sökmotorer att indexera webbplatsen och välj OK.
Om du tittar på adressen till en specifik sida har EPiServer gjort om adressen från www.foretaget.se/templates/page.aspx?id=1234 till www.foretaget.se/templates/page____1234.aspx. Det gör att sökmotorerna kan indexera sidan.
